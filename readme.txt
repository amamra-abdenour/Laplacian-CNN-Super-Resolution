Readme:

This project is about utilizing super resolution for image compression. Through the use of a Laplacian Super Resolution Network, images sizes were compressed to 4x their original size with good preservation of quality upon upsampling. 

Instructions to run:

1. Navigate to the home directory. 

2. run $ python tktest.py

Dependencies: some of these are os and gpu dependent. look at the docs for each on how to pip install.

pytorch
opencv
numpy
scipy io and scipy misc
tkinter
PIL
sys
math
os
time
